# MICE Uzbekistan

## Project

This is a Yii2 Basic application.

The goal is to build the National MICE Platform of Uzbekistan.

Never generate standalone HTML pages.

Always modify the existing Yii2 application.

Use MVC architecture.

Follow Yii2 best practices.

Use Bootstrap 5.

Never duplicate code.

Keep controllers thin.

Business logic belongs in models or services.

Always use ActiveRecord.

Use migrations.

Use Gii coding style.

Code must follow PSR-12.

Always keep the project maintainable.
